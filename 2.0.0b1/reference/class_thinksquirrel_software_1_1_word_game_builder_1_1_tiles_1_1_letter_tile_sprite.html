Detailed Description

Adds sprite display to a letter tile (Unity 4.3+ only).

This component must be on the same object as a letter tile.

Property Documentation

SpriteRenderer [] sprites
getset

The sprites to control with this component.

Sprite visibility is handled with the LetterTileVisibility component.
