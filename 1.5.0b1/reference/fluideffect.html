Fluid Effect (Legacy)

Overview

The Fluid Effect plugin creates a screen-space liquid metaball effect, using a combination of a component and a shader. This component requires a camera.

Note
Fluvio versions 2.2 and up contain new shaders that are incompatible with the Fluid Effect plugin. Therefore, it is recommended to use the new shaders rather than this effect.
See Also
FluidEffect

Properties

  • Plugin Type - Use this to change the plugin component type. You will need to reselect the game object after changing this property.
  • Fluid - The fluid associated with this plugin.
  • Cast Shadows - If enabled, the fluid will cast shadows. Note that shadows are not supported on mobile platforms.
  • Downsample Factor - When greater than 1, the final effect will be downsampled.
  • Skin Width - Additional amount of distance to add to the depth buffer. Increase this to deal with billboard clipping issues.
  • Opacity - The opacity of the fluid.
  • Threshold - The threshold opacity. Any pixels with an alpha greater than this value are shown.
  • Edge Power - The edge pixels have their opacity multiplied by the edge power.
  • Composite Color - The color to composite onto the screen with. This color is blended into the fluid at the end and does not take lighting into account.
  • Blur Iterations - If greater than 0, the fluid effect will be blurred in order to smooth harsh edges.
  • Blur Spread - The amount of blur for each blur iteration
  • Reference Point - The fluid effect supports having a second setting for at-distance view. The reference point provides a location to calculate the distance from.
  • Min Transition - After this distance, settings will transition to the distance settings.
  • Max Transition - After this distance, settings will be equal to the distance settings.
  • Min Distance - If a reference point is specified and the camera's distance from the reference point is greater than this value, the fluid effect will use the at-distance settings.
  • Distance Settings - Same as above, but will only take effect when the camera's distance from the reference point is greater than Min Transition.
