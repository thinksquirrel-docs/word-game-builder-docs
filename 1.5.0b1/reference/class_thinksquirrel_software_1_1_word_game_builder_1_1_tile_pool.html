Detailed Description

A class that manages an instance pool of letter tiles.

This class creates an instance pool based on a specific letter distribution.

void DestroyAllTiles ( )
inline

Destroys all tiles that were created by this tile pool (whether it is currently in the pool or not).

If a tile's parent object is changed, it will not be destroyed with this method and will become unmanaged by the pool.

void DestroyTile ( ILetterTile  tile)
inline

Destroys the tile from the scene, if the tile is within the pool.

This will fully destroy the tile and remove it from any player's possession. Beware - this is a destructive operation in edit mode!

int  count,
Vector3[]  positions,
Quaternion[]  rotations 
)
inline

Distributes and spawns the specified amount of tiles to the specified WordGamePlayer, up to WordGamePlayer.maxTiles (whichever comes first).

Count must be less than or equal to the length of both the position and rotation arrays. This automatically removes the tile from the tile pool, and gives them to the player (WordGamePlayer.heldTiles).
