Accessing Particles

Overview

Fluid particles can be accessed, similar to Shuriken. These particles can then be modified - manual forces can be assigned to these particles as well.

Example script

Here is an commented example script (C#) that gets particles, applies a random force, and then sets them again.

using ThinksquirrelSoftware.Fluvio;
public class GetParticlesExample : MonoBehaviour
{
public Fluid myFluid;
public float force = 20;
FluidParticle[] myParticles;
void OnEnable()
{
// Subscribe to the pre-solve event
myFluid.OnPreSolve += OnPreSolve;
}
void OnDisable()
{
// Some cleanup
myFluid.OnPreSolve -= OnPreSolve;
}
void OnPreSolve()
{
// Get the current set of particles. This will automatically create or resize the particle array
myFluid.GetParticles(ref myParticles);
for(int i = 0; i < myParticles.Length; ++i)
{
// We need to assign the particle struct to a temporary variable because it is a value type
FluidParticle particleTemp = myParticles[i];
// Add a random force to the particle. We're going to ignore mass by specifying the acceleration force mode (optional)
particleTemp.AddForce(Random.onUnitSphere * force, ForceMode.Acceleration)
// Place the struct back into the array
myParticles[i] = particleTemp;
}
// Send the modified particles back to the fluid
myFluid.SetParticles(myParticles, myParticles.Length);
}
}
